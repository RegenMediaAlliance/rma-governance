# Pathway Ahead

Where RMA governance goes from here. This is the living roadmap the alliance drives
via pull request. A clickable, color-coded version lives on the RMA Notion board
(the "Build Cockpit").

## Where we are — Release 0 (this repo)
A thin, honest baseline: the Charter spine, RMA's entity + structure (independent,
aligned with Quorum 1, point + triad), the three-role methodology framed as peers, the
adopted Q-Git change-flow, and the priority gaps marked `[OPEN]`. Published as-is for
member feedback before the Wednesday call.
